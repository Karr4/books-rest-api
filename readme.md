## Books Rest Api

### Tech Stack

- Node.js
- Express.js
- MongoDB

Run 'npm run start' to start project in production mode

```bash
$ npm run start
```

Run 'npm run start:dev' to start project in development mode

```bash
$ npm run start:dev
```
